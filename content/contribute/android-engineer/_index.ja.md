### **Organic Maps について**

**Organic Maps** は、旅行者、ハイカー、ライダー、アウトドア愛好家のためのシンプルで無駄のないオフライン地図・ナビゲーションアプリです。地図、検索、ナビゲーションは完全にオフラインで動作し、通信電波は不要です。広告も、トラッカーも、データ収集も一切なく、プライバシーだけを尊重します。オープンソース愛好家によって開発され、OpenStreetMap のデータを利用し、世界中のコミュニティに支えられています。

Organic Maps は [2024年末時点で](@/news/2024-12-20/411/index.md):

- 世界中の国々から **約300万人のユーザー**
- AppStore と Google Play で **約15,000件の評価とレビュー**
- AppStore / Google Play の **平均評価 4.8 / 4.6**
- GitHub で **10,000以上のスター**
- GitHub で **約10,000件の issue と PR**
- GitHub で **約1,000件のフォーク**
- **100名以上の貢献者による 7,000件以上の git コミット**
- **マーケティング費は $0** - 純粋なオーガニック成長

### ビジョン

私たちのビジョンは、プライバシーを組み込んだ、コミュニティが開発する自由でオープンなナビゲーションを提供することです。Organic Maps は、ビッグテック、VC、銀行の利益ではなく、ユーザーのニーズを優先し、公共の利益に資するナビゲーションアプリの構築を目指しています。

### 機会

プライバシーとオフライン機能の新たな基準を打ち立てる、世界トップクラスのナビゲーションアプリの構築に貢献してください。企業の利害から独立したこのアプリの未来を形作る上で、あなたは重要な役割を担います。旅行者にとってのナビゲーションの未来を形作る旅路に参加してください。

### 担当業務

- Android アプリの新機能を設計から本番投入まで開発・実装する。
- 大規模に効率よく動作する技術ソリューションを設計する。
- NDK を介した C++ ライブラリの統合を含む、Android (Java) のコードを記述する。
- プロダクト、デザイン、エンジニアリング、サポートと協働し、直感的でユーザーフレンドリーな体験を作り上げる。
- 最新の Android ガイドラインを常に把握し、アプリのコンプライアンスを維持する。
- アプリ の安定性とパフォーマンスを継続的に向上させる。
- 世界中のオープンソースコミュニティと協力して開発を推進する。
