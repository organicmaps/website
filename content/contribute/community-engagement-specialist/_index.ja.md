### Organic Maps について

Organic Maps は、ドライバー、ハイカー、ライダーのためのプライバシーに焦点を当てた、コミュニティ開発のナビゲーションアプリです。プライバシー保護を備えたナビゲーション機能を備え、位置情報の追跡、データ収集、広告は一切行いません。検索、ルーティング、ナビゲーションは携帯電話の電波なしで動作するため、遠隔地のハイキングコースや通信状況が悪い場所でも最適です。Organic Maps は世界中に貢献者がいるクラウドソースの地図データ OpenStreetMap を利用しています。Organic Maps はコミュニティ主導の開発とコラボレーションを重視しています。

### ビジョン

地図を使った旅が、プライバシー重視をデフォルトとし、地球上で最も選ばれる選択肢となる世界を創ります。コミュニティの力によって、プライバシー重視の地図で自由にナビゲートできる未来を解き放ちます。

### 機会

コミュニティエンゲージメントスペシャリストとして、Organic Maps を利用するコミュニティの人々と直接関わり、アプリとウェブサイトに関するテクニカルサポートと支援を提供します。コミュニティは技術に精通したエンジニアから、テクノロジーに不慣れなユーザーまで多岐にわたります。サポート業務には、高い満足度の維持、技術的な課題の改善、より良い全体的なナビゲーション体験を提供するためのサポート体験の強化が含まれます。

### 担当業務

- メール、App Store、Google Play、Telegram を通じて世界中のユーザーにサポートを提供する。
- iOS と Android の技術的な問題を調査、トラブルシューティング、診断する。
- 製品と GitHub 上の現在の issue について深い理解を維持する。
- 問題の解決策についてコミュニケーションし、ガイダンスを提供する。
- バグを評価し、再現し、文書化する。
- 繰り返し発生するリクエストやバグの統計を GitHub Issues にまとめ、フィードバックやインサイトをコミュニティの貢献者にフィードバックする。
- プロダクト、デザイン、エンジニアリング、その他 Organic Maps コミュニティ内の貢献者と協力して問題に対処する。
- 製品機能と FAQ のためのナレッジベース記事を作成する。
- スケーリング、問題処理、レポート、分析を含むサポートプロセスを継続的に改善し、ユーザーのサポート体験を向上させる。
