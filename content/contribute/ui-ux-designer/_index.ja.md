**雇用形態**: パートタイム、プロジェクトベース。

### 概要

Organic Maps は、ドライバー、ハイカー、ライダーのためのプライバシーに焦点を当てた、コミュニティ開発のナビゲーションアプリです。プライバシー保護を備えたナビゲーション機能を備え、位置情報の追跡、データ収集、広告は一切行いません。検索、ルーティング、ナビゲーションは携帯電話の電波なしで動作するため、遠隔地のハイキングコースや通信状況が悪い場所でも最適です。Organic Maps は世界中に貢献者がいるクラウドソースの地図データ OpenStreetMap を利用しています。Organic Maps はコミュニティ主導の開発とコラボレーションを重視しています。

### ビジョン

地図を使った旅が、プライバシー重視をデフォルトとし、地球上で最も選ばれる選択肢となる世界を創ります。コミュニティの力によって、プライバシー重視の地図で自由にナビゲートできる未来を解き放ちます。

### 機会

Organic Maps の UI/UX デザイナーとして、私たちのビジョンに沿い、世界中のコミュニティのニーズを満たす、魅力的で美しいモバイル体験を形作る重要な役割を担います。主な業務は、iOS と Android プラットフォーム向けの、視覚的に美しく直感的なユーザーインターフェースのデザインとプロトタイピングです。ワイヤーフレーム、ユーザーフロー、モックアップ、インタラクティブなプロトタイプの作成や、社内外のチームとのコラボレーションを通じてアイデアを具現化することが含まれます。デザインシステムの構築と維持、業界トレンドの先行把握、チームとコミュニティからのフィードバックに基づくアプリデザインの最適化も担当します。美的感覚と機能性のバランスを取りながら、デザイン戦略全体を形作り、喜ばれる製品を提供することに寄与します。すべての人の旅とナビゲーションをより良いものにする体験をデザインする、永続的なインパクトを与えるユニークな機会です。

### 担当業務

- Organic Maps のビジョンとユーザーのニーズに合致した、**直感的で視覚的に魅力的なユーザーインターフェース**をモバイル向けにデザインしプロトタイプ化する。
- **ワイヤーフレーム、ユーザーフロー、モックアップ、インタラクティブなプロトタイプ**を作成し、デザインアイデアをステークホルダーに効果的に伝える。
- **デザインシステムとスタイルガイド**を構築・維持し、すべてのモバイルプラットフォームと機能で一貫性を確保する。
- **iOS と Android の両プラットフォーム向けにデザイン**し、Material Design や Apple Human Interface Guidelines などのプラットフォーム固有のガイドラインを遵守する。
- **デザインコンセプトとプロトタイプをステークホルダーに提示**し、フィードバックを最終デザインに反映する。
- プロダクトマネージャー、開発者、コミュニティを含む**クロスファンクショナルチームとコラボレーション**し、目標を魅力的な体験に変換する。
- モバイル UI/UX デザインの**最新の業界トレンド、ツール、ベストプラクティスを常に把握**し、革新的なアイデアを提案する。
- フィードバックに基づく**反復的な改善**を通じて、モバイルアプリデザインを最適化する。
- モバイルアプリが視覚的に美しく、かつ高い機能性を備えるよう、**美的感覚と機能性のバランスを取る**。
- アクセシビリティ標準に準拠し、**すべての人が使いやすいアクセシブルな体験**をデザインする。
- 製品全体の**デザイン戦略とビジョンに貢献**し、製品の長期目標とユーザー満足を支える。
