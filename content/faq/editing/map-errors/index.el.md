Η πηγή δεδομένων χάρτη μας είναι [OpenStreetMap](https://www.openstreetmap.org/) (OSM). Αυτό είναι ένα έργο χαρτογράφησης παρόμοιο με τη Wikipedia, αλλά για χάρτες, όπου ο καθένας μπορεί να δημιουργήσει και να επεξεργαστεί τον χάρτη του κόσμου.

Εάν δείτε λανθασμένες πληροφορίες ή εάν διαπιστώσετε ότι λείπουν ορισμένα αντικείμενα στον χάρτη, μπορείτε να [αφήσετε μια σημείωση](https://www.openstreetmap.org/note/new) για τους εθελοντές του OSM ή να [εγγραφείτε](https://www.openstreetmap.org/user/new) και να επεξεργαστείτε τον χάρτη.

Όσο περισσότεροι άνθρωποι συνεισφέρουν, τόσο πιο λεπτομερείς χάρτες παίρνουν όλοι. Πιστεύουμε ότι ο πιο λεπτομερής χάρτης όλου του κόσμου, που δημιουργήθηκε από την ανοιχτή κοινότητα, είναι απλώς θέμα χρόνου.

**Σημειώσεις:**

* Μπορείτε επίσης να προσθέσετε νέα μέρη, να επεξεργαστείτε υπάρχοντα POI και πληροφορίες κτιρίων (διευθύνσεις, ώρες λειτουργίας, ονόματα) απευθείας στους Οργανικούς Χάρτες. Μόλις συνδεθείτε με λογαριασμό OSM, οι αλλαγές σας θα μεταφορτωθούν αυτόματα στο OSM. Να είστε προσεκτικοί κατά την επεξεργασία, καθώς οι αλλαγές σας θα είναι ορατές σε όλους τους άλλους χρήστες.

* Η βάση δεδομένων OpenStreetMap ενημερώνεται κάθε λεπτό. Στόχος μας είναι να ενημερώνουμε χάρτες στην εφαρμογή 1-4 φορές το μήνα. Εάν τροποποιήσατε κάτι στο OSM, οι αλλαγές σας θα εμφανίζονται σε μελλοντικές ενημερώσεις χαρτών.
