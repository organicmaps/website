Για να εγκαταστήσετε το OM σε Linux με flatpak, ανοίξτε το τερματικό και πληκτρολογήστε "flatpak install flathub app.organicmaps.desktop"

Μόλις εγκατασταθεί η εφαρμογή, μπορείτε να χρησιμοποιήσετε τον τροχό κύλισης του ποντικιού σας ή τα χειριστήρια στη δεξιά γραμμή μενού για να κάνετε μεγέθυνση στην περιοχή που θέλετε να πλοηγηθείτε για να κατεβάσετε χάρτες για αυτήν την περιοχή. Μπορείτε επίσης να κάνετε κλικ στο εικονίδιο "λήψη" κάτω δεξιά. Αφού κατεβάσετε χάρτες για τις περιοχές που σας ενδιαφέρουν, η εφαρμογή θα πρέπει να λειτουργεί ακόμα και χωρίς σύνδεση στο διαδίκτυο. 

Μπορείτε να τοποθετήσετε το ποντίκι σας πάνω από τα διάφορα εικονίδια για να δείτε κάποιο κείμενο βοήθειας. 

Για να εκτελέσετε δρομολόγηση και περιήγηση με στροφή, έχετε μερικές επιλογές. εάν γνωρίζετε τις συντεταγμένες GPS των σημείων αφετηρίας και λήξης, μπορείτε να κάνετε κλικ στο εικονίδιο ρυθμίσεων (πάνω από το πράσινο σημάδι επιλογής) και να εισαγάγετε τις συντεταγμένες του σημείου εκκίνησης και του προορισμού σας. Για να ορίσετε το σημείο εκκίνησης στον χάρτη, κάντε κλικ στο εικονίδιο πλοήγησης και επιλέξτε "σημείο έναρξης", στη συνέχεια κρατήστε πατημένο το πλήκτρο shift και κάντε αριστερό κλικ στον χάρτη. Για να ορίσετε τον προορισμό, αλλάξτε σε "τελικό σημείο" και κάντε κλικ σε μια τοποθεσία στο χάρτη.

![Ρύθμιση σημείου έναρξης και σημείου λήξης](nav_points.png)

Μπορείτε να κάνετε κλικ στο μπλε εικονίδιο ακριβώς πάνω από το εικονίδιο ρυθμίσεων για να διαγράψετε την πλοήγηση. 

![εκκαθάριση πλοήγησης](clear_route.png)

Για να αναζητήσετε διευθύνσεις και προορισμούς, κάντε κλικ στον μεγεθυντικό φακό και εισαγάγετε τη διεύθυνση ή τον όρο αναζήτησης.

Για να προσθέσετε σελιδοδείκτη σε μια τοποθεσία, κρατήστε πατημένο το πλήκτρο Alt και κάντε δεξί κλικ στο μέρος που θέλετε να προσθέσετε σελιδοδείκτη. Ο σελιδοδείκτης ενδέχεται να μην είναι άμεσα ορατός, για να προβάλετε και να διαχειριστείτε τους σελιδοδείκτες πρέπει να κάνετε κλικ στο εικονίδιο με το κόκκινο αστέρι. 

![προβολή σελιδοδεικτών](view_bookmarks.png)

Η εφαρμογή για επιτραπέζιους υπολογιστές Linux χρησιμοποιείται κυρίως για σκοπούς ανάπτυξης (αυτόματη δοκιμή και έλεγχος λογικής χωρίς να χρειάζεται μεταγλώττιση για κινητά). Οποιοσδήποτε εθελοντής να βελτιώσει τη χρηστικότητα της έκδοσης Linux είναι ευπρόσδεκτος!
