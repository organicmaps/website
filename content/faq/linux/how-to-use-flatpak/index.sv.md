För att installera OM på Linux med flatpak, öppna terminalen och mata in `flatpak install flathub app.organicmaps.desktop`

När appen är installerad kan du använda musens rullningshjul eller kontrollerna i den högra menyraden för att zooma in på det område du vill navigera för att ladda ner kartor för det området. Du kan också klicka på "ladda ner"-ikonen nere till höger. När du har laddat ner kartor för de regioner du bryr dig om ska appen fungera även utan internetuppkoppling. 

Du kan föra musen över de olika ikonerna för att se lite hjälptext. 

För att utföra routing och sväng för sväng-navigering har du ett par alternativ. om du känner till GPS-koordinaterna för dina start- och slutpunkter kan du klicka på inställningsikonen (ovanför den gröna bocken) och ange koordinaterna för din startpunkt och destination. För att ställa in startpunkten på kartan, klicka på navigeringsikonen och välj "startpunkt", håll sedan ned shift-tangenten och vänsterklicka på kartan. För att ställa in destinationen, ändra till "slutpunkt" och klicka på en plats på kartan.

![Ställa in startpunkt och slutpunkt](nav_points.png)

Du kan klicka på den blå ikonen direkt ovanför inställningsikonen för att rensa navigeringen. 

![rensa navigering](clear_route.png)

För att söka efter adresser och destinationer, klicka på förstoringsglaset och ange adressen eller söktermen.

För att bokmärka en plats håller du ned Alt-tangenten och högerklickar på den plats du vill bokmärka. Bokmärket kanske inte syns direkt, för att visa och hantera bokmärken måste du klicka på den röda stjärnikonen. 

![visar bokmärken](view_bookmarks.png)

Linux-skrivbordsappen används mest för utvecklingsändamål (automatisk testning och kontrolllogik utan att behöva kompilera för mobil). Alla frivilliga för att förbättra användbarheten av Linux-versionen är välkomna!
