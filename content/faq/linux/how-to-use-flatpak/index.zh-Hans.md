要使用 flatpak 在 Linux 上安装 OM，请打开终端并输入“flatpak install flathub app.organicmaps.desktop”

安装应用程序后，您可以使用鼠标滚轮或右侧菜单栏上的控件放大您想要导航的区域，以便下载该区域的地图。您也可以点击右下角的“下载”图标。下载您关心的地区的地图后，即使没有互联网连接，该应用程序也应该可以运行。 

您可以将鼠标悬停在各个图标上以查看一些帮助文本。 

要执行路线选择和路线导航，您有多种选择。如果您知道起点和终点的 GPS 坐标，可以单击设置图标（绿色对勾上方）并输入起点和终点的坐标。要在地图上设置起点，请单击导航图标并选择“起点”，然后按住 Shift 键并在地图上单击鼠标左键。要设置目的地，请更改为“终点”并单击地图上的位置。

![设置起点和终点](nav_points.png)

您可以单击设置图标正上方的蓝色图标来清除导航。 

![清晰导航](clear_route.png)

要搜索地址和目的地，请单击放大镜并输入地址或搜索词。

要为某个位置添加书签，请按住 Alt 键并右键单击要添加书签的位置。书签可能不会立即可见，要查看和管理书签，您需要单击红星图标。 

![查看书签](view_bookmarks.png)

Linux 桌面应用程序主要用于开发目的（自动测试和检查逻辑，无需为移动设备进行编译）。欢迎任何致力于提高 Linux 版本可用性的志愿者！
