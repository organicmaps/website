## Release Notes

- *NEW!* Highlighted downloaded regions on the World map (Viktor Govako)
- *NEW!* See and search for nature reserves, national parks, protected areas, Aboriginal lands, danger areas, and lakes on the World map starting at zoom level 8 (Viktor Govako)
- *NEW!* Routing now supports road closure times (Viktor Govako)

### All platforms

- OpenStreetMap data as of November 9, 2025 (Viktor Govako)
- Fixed an issue with default and localized street names when editing addresses and uploading changes to OpenStreetMap (Viktor Govako)
- Fixed the zoom level applied when tapping on search results from the World map (Viktor Govako)
- Fixed cases where the navigation autozoom setting did not work (Viktor Govako)

### iOS

- After deleting a bookmarks list or tracks list (via the "More" button), the app now correctly returns to the previous screen (Kiryl Kaveryn)
- Added [TikTok][tiktok], [Threads][threads], and [Bluesky][bluesky] social media links to the "About" screen (Kiryl Kaveryn)
- Added "Report a bug" to the app icon's quick actions on the home screen (Kiryl Kaveryn)
- Fixed various app and CarPlay crashes (Kiryl Kaveryn)

### Android

- Updated Text-to-Speech (TTS) settings (Andrei Shkrob)
- Fixed an Android Auto crash (Andrei Shkrob)
- Android Auto always uses the car routing mode (Andrei Shkrob)
- Use Traditional Chinese translations for Hong Kong (zh-HK) and Macau (zh-MO) regions (Alexander Borsuk)

Please let us know if you can run Organic Maps on any x86 devices (including Chromebooks) with OpenGL ES 3 support.

### Map & Styles

- Swapped the rendering of 'abandoned' and 'disused' railways (Lukas Hamm)
- Added watchmakers and student accommodations to the map (David Martinez)
- Added travel agency icons (David Martinez)
- Changed the toilet icon color to gray (David Martinez)
- Updated the audiologist, podiatrist, optometrist, and hearing aid icons (David Martinez)
- Fixed the bicycle rental icon in dark mode (David Martinez)
- Display U.S. and Canadian state names earlier on the World map (from zoom level 5) (Viktor Govako)

### Translations

- Added Slovenian search synonyms and enabled category search in Slovenian (Alexander Borsuk)
- Fixed typos, incorrect translations, and search synonyms, and added missing translations for Arabic, Bulgarian, Danish, German, Greek, Spanish, Persian (Farsi), Finnish, Hungarian, Indonesian, Japanese, Korean, Lithuanian, Latvian, Norwegian Bokmål, Dutch, Romanian, and Slovenian (Alexander Borsuk)
- Added dedicated translations for Mexican Spanish (es-MX) and Brazilian Portuguese (pt-BR), distinct from standard Spanish (es) and Portuguese (pt) (Alexander Borsuk)
- Corrected Czech TTS voice directions (jxsv)
- Enabled TTS voice directions for Estonian (et), Galician (gl), Hebrew (he), and Lithuanian (lt) (Sergiy Kozyr)
- Updated translations (Sergiy Kozyr, Weblate contributors)

### Desktop

- Show World map features — such as nature reserves and lakes — in the desktop context menu (Viktor Govako)
- Added a setting to enable or disable highlighting of downloaded regions on the map (Viktor Govako)
