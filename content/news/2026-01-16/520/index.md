Let's start 2026 with an Organic Maps update, made possible by all our contributors! Install it from <https://omaps.app/get> or from [App Store][appstore], [Google Play][googleplay], [Huawei AppGallery][appgallery], [Obtainium][obtainium], [Accrescent][accrescent], and [F-Droid][fdroid]

### Release notes (all platforms)

* **Higher-contrast dark theme colors** (David Martinez)
* Updated OpenStreetMap data (January 11) (Viktor Govako)
* Added Albanian voice guidance (TTS) (Alexander Borsuk)
* Added Bulgarian search synonyms (Clippy)
* Excluded casinos from the "Entertainment" category (Clippy)
* Reduced the thickness of subway lines (Manik)
* Added capacity information to search results for motorcycle parking, bicycle rental, bicycle charging, and car charging (David Martinez)
* Added floor level to search results (David Martinez)

### Translations

* Added Albanian app translations (Alexander Borsuk)
* Updated the FAQ and added FAQ translations for Arabic, Czech, Dutch, Farsi, Greek, Hebrew, Hindi, Hungarian, Indonesian, Italian, Lithuanian, Marathi, Swedish, Telugu (Alexander Borsuk)
* Improved German app and FAQ translations (DaN0mic)
* Improved Latin American Spanish app translations (Alexander Borsuk)
* Renamed the "Zoom buttons" setting for clarity (Alexander Borsuk)
* Translation fixes from Weblate contributors for various languages (Viktor Govako)

### iOS only changes

* Wider route planning buttons (Kiryl Kaveryn)

### Android only changes

* Added Google Assistant support for navigation and search (skirm-tech)
* Updated the "Auto" navigation theme setting to follow the system dark/light mode (Dzmitry Strekha)
* Fixed a crash when starting navigation (Owm Dubey)
* Fixed a crash when uploading notes to OpenStreetMap (Prakamy Awasthi)
* Fixed a crash in Android Auto (Alexander Borsuk)
* Fixed missing TTS voice guidance announcements (Alexander Borsuk)
