Time to update Organic Maps! Faster cross-country and cross-region routes, fresh OSM map data, and many bug fixes for Android in the second app update in 2026.

Install it from [get.omaps.org](https://get.omaps.org) or from [App Store][appstore], [Google Play][googleplay], [Huawei AppGallery][appgallery], [Obtainium][obtainium], [Accrescent][accrescent], and [F-Droid][fdroid].

### What's New

* Routes between countries are created faster and no longer cause out-of-memory crashes in some cases _(Viktar Havaka)_
* Updated OpenStreetMap data as of January 25 _(Viktar Havaka)_
* GeoJSON export support for bookmarks and tracks _(Sergiy Kozyr, Alexander Borsuk, Viktar Havaka)_
* Fixed and updated links to the Organic Maps website _(Alexander Borsuk)_

### Android

* Expanded opening hours to show full-week hours _(rovertrack)_
* Split the Light/Dark app theme option into Appearance and Navigation settings _(Dzmitry Strekha)_
* Tapping a bookmark icon in the bookmarks list allows changing its color _(Abdelhadi Wael)_
* Improved landscape UI for a selected place on the map _(Rover track)_
* Fixed a non-working editor for places with invalid phone numbers _(Abdelhadi Wael)_
* Fixed a crash on Android 12 devices _(Alexander Borsuk)_
* Fixed long street name clipping in the navigation header _(pranjaldubey022)_
* Fixed routing toolbar overlap on small screens _(Anton P)_
* Fixed a search crash on Android 5 and 6 _(Alexander Borsuk)_
* Fixed FAQ translations on Android 5 _(Alexander Borsuk)_
* Fixed a “Not Enough Free Space” error when maps are updated _(Alexander Borsuk)_

### Translations

* Fixed and updated translations from Weblate contributors _(Viktar Havaka)_
* Updated Romanian TTS voice directions _(R. Ravzan)_
* Improved countries descriptions _(alnzrv)_
* Fixed Romanian and Mexican Spanish translations _(Alexander Borsuk)_
* Added missing translations for disused bridges _(Alexander Borsuk)_
