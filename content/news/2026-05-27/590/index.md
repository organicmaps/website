Meet another Organic Maps May update with fixes and improvements (more details below). Organic Maps team is grateful to everyone who supports the project, [donates](@/donate/index.md), spreads the word, writes good reviews and [contributes](@/contribute/index.md). More new features to come, including improved [public transport](@/news/2026-05-08/580/index.md), stay tuned!

Get the update at <https://get.omaps.org> or on the [App Store][appstore], [Google Play][googleplay], [Huawei AppGallery][appgallery], [Obtainium][obtainium], [Accrescent][accrescent], and [F-Droid][fdroid].

### All platforms

- OSM map data as of May 27, 2026 _(Viktor Govako)_
- Added icons for boules, cycling, and motorsport places _(Viktor Govako)_
- Added ceremonial gates, water slides, theme parks, and roller coasters _(David Martinez)_
- Fixed Australia and New Zealand road shields _(Viktor Govako)_
- Improved finger gestures for map control in perspective view during Navigation mode _(Viktor Govako)_
- Fixed public transport routes for stations and other area objects (tap on a station or stop to see routes) _(Viktor Govako)_
- Fixed rendering of speed limits for speed cameras _(Alexander Borsuk)_
- Fixed antimeridian zoom-out with more than one world copy _(Alexander Borsuk, Viktor Govako)_
- More accurate display of street and house addresses _(Viktor Govako)_
- Fixed unnecessary map animations _(Mikhail Listratsenka)_
- Stability and performance improvements _(Alexander Borsuk, Viktor Govako)_

### Translations

- Improved French translations _(Meenphie)_
- Improved Italian translations _(Andrea Pappacoda)_
- Improved Spanish translations _(Jesús L. Pinzolas)_
- Improved Ukrainian translations _(Nnifria)_
- Improved Catalan translations, updated translations for several country and region names, fixed search synonyms for `leisure=track` _(Alexander Borsuk)_
- Corrected translations for `disused:bridge` and `disused:tunnel` _(alnzrv)_

### iOS

- NEW! Track text and HTML descriptions are now supported, the same way as descriptions for bookmarks. Press the pencil icon to add a description. _(Kiryl Kaveryn)_
- Text-to-Speech (TTS) now picks the best installed voice for the language (Enhanced / Premium). See Organic Maps settings for how to download a higher-quality voice. _(Kiryl Kaveryn)_
- Fixed a track-renaming bug where the new name was not properly updated _(Kiryl Kaveryn)_
- CarPlay: fixed the recenter button _(Kiryl Kaveryn)_
- Fixed crash when adding a bookmark on a hiking or cycling route _(Alexander Borsuk)_

### Android
- NEW! Track text and HTML descriptions are now supported, the same way as descriptions for bookmarks. Press the pencil icon to add a description. _(Mikhail Listratsenka)_
- NEW! Choose any color for tracks, not only a limited subset _(Mikhail Listratsenka)_
- Korean, Chinese, and Japanese users will now see proper fonts based on device locale _(Alexander Borsuk)_
- Fixed track place-page issues and "Open In" button _(Mikhail Listratsenka)_

### Linux and macOS
- Added Chinese translations to the `.desktop` file comments _(DeepChirp)_
